# v6.1.2

Release date: 2026-07-02

## Summary

This patch release delivers the requested motif workflow refinements for iPad usability by moving motif selection into a dedicated modal while keeping symmetry controls always visible in the top toolbar.

## Highlights

- Replaced the inline tabbed motif panel with a dedicated Motifs modal.
- Kept the Symmetry dropdown permanently available in the main top bar.
- Added robust modal close behavior: close button, backdrop click, and Escape key.
- Removed the Blank motif option now that Clear Canvas already handles reset flow.
- Updated the Moon motif to a cleaner crescent-and-stars design.

## Included Updates

- Topbar + modal markup updates in `index.html`.
- Modal layout and responsive sizing in `features/studio-ui/studio-ui.css`.
- Modal interaction and toolbar wiring updates in `features/studio-ui/studio-ui.js`.
- Moon motif definition update in `features/project-core/project-core.js`.
- E2E workflow updates in `features/studio-e2e/studio-e2e.test.cjs`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Modal uses dialog semantics (`role="dialog"`, `aria-modal="true"`, and title association).
- Keyboard close affordance provided through Escape key support.
- Existing focus-visible styling remains active for keyboard users.

- `npm run test` passed.
