# v6.1.11

Release date: 2026-07-03

## Summary

This patch release expands motif composition for taller canvases by adding a full-window tarot-inspired set designed specifically for 19x26 layouts and aligning those motifs to core project symbolism.

## Highlights

- Added three full-window motifs intended for 19x26 available space and vertical card framing.
- Reframed the new set with Major Arcana naming and symbolism:
  - `XVI • The Tower Reforged`
  - `XVII • The Star Lotus`
  - `X • Wheel Of Fortune`
- Preserved stable motif IDs for compatibility while deepening tarot-card style accents.
- Added core test assertions confirming the new full-window motifs are present in the template library.

## Included Updates

- Added and tuned full-window motif definitions in `features/project-core/motifs/single-motifs.js`.
- Added motif presence checks in `features/project-core/project-core.test.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.
