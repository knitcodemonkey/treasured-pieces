# v6.1.5

Release date: 2026-07-02

## Summary

This patch release adds variable canvas width and height controls so makers can quickly resize the drawing surface to fit different pattern needs.

## Highlights

- Added toolbar controls for canvas width and height with a one-click resize action.
- Added keyboard-friendly apply behavior (Enter key) for dimension inputs.
- Rebuilt symmetry and motif preview behavior on resize so interactions stay aligned.
- Added project-core resize support with guardrails and template regeneration.
- Expanded automated test coverage for both core and end-to-end resize flows.

## Included Updates

- Updated controls markup in `index.html`.
- Added sizing control styles in `features/studio-ui/studio-ui.css`.
- Added resize orchestration and canvas resync logic in `features/studio-ui/studio-ui.js`.
- Added `resize` support in `features/project-core/project-core.js`.
- Added resize assertions in `features/project-core/project-core.test.js`.
- Added end-to-end resize scenario in `features/studio-e2e/studio-e2e.test.cjs`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- New width/height controls are labeled and keyboard operable.
- Resize action is available via button click and Enter key in input fields.
- Existing focus-visible styles continue to apply to new controls.
