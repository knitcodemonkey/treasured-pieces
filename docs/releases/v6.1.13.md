# v6.1.13

Release date: 2026-07-03

## Summary

This patch release improves tall-card clarity by replacing the less-readable full-window Star motif with two stronger silhouette-first major-arcana cards: Sun and Moon.

## Highlights

- Removed the full-window Star motif that was visually ambiguous at low pixel density.
- Added two clear 19x26 full-window motifs:
  - `XIX • The Sun` (`full-window-sun-arcana`)
  - `XVIII • The Moon` (`full-window-moon-arcana`)
- Preserved `X • Wheel Of Fortune` unchanged as the preferred full-window tarot reference.
- Updated core test assertions to cover the new motif IDs.

## Included Updates

- Replaced motif definition set in `features/project-core/motifs/single-motifs.js`.
- Updated template presence assertions in `features/project-core/project-core.test.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- This release updates motif artwork and metadata only; no interaction model changes were introduced.
