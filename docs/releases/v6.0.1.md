# v6.0.1

Release date: 2026-07-02

## Summary

This patch release improves iPad mini and Apple Pencil usability, focusing on more intuitive toolbar flow, cleaner tooltip behavior, and smoother stylus drawing interactions.

## Highlights

- Refined toolbar and control ergonomics for tablet-first use.
- Improved Pencil drawing continuity with cell interpolation during faster strokes.
- Added a Pencil gesture to toggle quickly between active paint color and white erase.
- Removed duplicate native browser tooltip overlays on swatches.
- Repositioned custom palette tooltips above swatches with viewport-aware fallback.

## Included Updates

- Updated studio UI control and tooltip behavior in `features/studio-ui/studio-ui.js`.
- Updated tooltip placement styling and touch ergonomics in `features/studio-ui/studio-ui.css`.
- Updated toolbar copy and control grouping polish in `index.html`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Tooltip layering conflict reduced by removing native `title` hover duplication.
- Custom tooltip positioning now reduces occlusion risk for stylus users.
- CI accessibility policy remains unchanged: AA required, AAA advisory.

- `npm run test:core` passed.
- `npm run test:e2e` passed.
