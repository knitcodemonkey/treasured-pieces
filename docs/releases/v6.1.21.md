# v6.1.21

Release date: 2026-07-04

## Summary

This patch release improves Build Counts usability by making the panel collapsible and removing zero-value rows so only relevant material counts are shown.

## Highlights

- Converted Build Counts into a native accordion that can be expanded or collapsed.
- Kept Build Counts visible by default while allowing quick hide/show control.
- Filtered out zero-count colors from the count list for a cleaner, more actionable view.

## Included Updates

- Updated Build Counts markup to use an accordion container in `index.html`.
- Added accordion header/toggle styling in `features/studio-ui/studio-ui.css`.
- Updated color usage rendering to exclude zero-count entries in `features/studio-ui/studio-ui.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.

## Accessibility Compliance

- Uses native HTML `details/summary` semantics for toggle behavior.
- This release introduces no keyboard-breaking interaction changes.
