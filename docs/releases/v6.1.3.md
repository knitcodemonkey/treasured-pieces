# v6.1.3

Release date: 2026-07-02

## Summary

This patch release refines top-bar ergonomics by decluttering non-essential controls, clarifying feature grouping, and separating destructive actions for safer day-to-day use.

## Highlights

- Hid the Grid Overlay toggle from the top bar for a cleaner primary workflow.
- Set grid visibility default to off at startup.
- Tightened Symmetry label/dropdown spacing to reduce control ambiguity.
- Split toolbar into feature controls and destructive actions.
- Finalized alignment: Motifs then Symmetry on the left, Clear Canvas on the right.

## Included Updates

- Top-bar structure and version label updates in `index.html`.
- Toolbar alignment/grouping updates in `features/studio-ui/studio-ui.css`.
- Optional grid-toggle handling in `features/studio-ui/studio-ui.js`.
- Grid default visibility update in `features/project-core/project-core.js`.
- Baseline/version updates in `package.json`, `CHANGELOG.md`, and `README.md`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Clear Canvas remains separately grouped to reduce accidental destructive activation.
- Focus-visible behavior remains unchanged for keyboard navigation.
- Control label association remains explicit for form fields.

- `npm run test` passed.
