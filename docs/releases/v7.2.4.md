# v7.2.4

Release date: 2026-07-04

## Summary

This patch release refines the studio information hierarchy by making the palette bar itself the Build Counts accordion title, with counts content expanding directly beneath it.

## Highlights

- Moved the palette into the accordion summary row so color selection remains visible while counts are collapsed.
- Added right-side Build Counts title and live summary metadata in the summary row.
- Kept expand/collapse behavior intuitive with a clear chevron indicator and downward content reveal.
- Improved top toolbar wrapping behavior to better support desktop and iPad portrait workflows.

## Included Updates

- Updated `index.html` to use a palette-first accordion summary structure (`palette-accordion` and `palette-header`).
- Updated `features/studio-ui/studio-ui.css` with palette-accordion header/layout rules and cleanup for valid CSS structure.
- Updated `features/studio-e2e/studio-e2e.test.cjs` selectors to match the new accordion DOM and interaction target.
- Updated release metadata in `package.json`, `package-lock.json`, `README.md`, `CHANGELOG.md`, `docs/releases/README.md`, and `index.html`.
