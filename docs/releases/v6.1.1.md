# v6.1.1

Release date: 2026-07-02

## Summary

This patch release stabilizes iPad toolbar behavior by reducing mode complexity, keeping critical actions always visible, and ensuring the color bar fits the current 16-color workflow without wrapping or side-scroll.

## Highlights

- Grid and Clear controls are now always visible in the top toolbar.
- Simplified feature tabs to Symmetry and Motifs.
- Reduced page movement by stabilizing control-panel behavior.
- Tuned palette layout so all 16 colors fit in a single bar on iPad.
- Added roadmap wording for future curated palette bar support.

## Included Updates

- Toolbar and panel structure updates in `index.html`.
- Layout and palette sizing updates in `features/studio-ui/studio-ui.css`.
- Toolbar default panel behavior updates in `features/studio-ui/studio-ui.js`.
- E2E flow update for always-visible clear action in `features/studio-e2e/studio-e2e.test.cjs`.
- Roadmap update in `docs/roadmap.md`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Focus-visible styles remain in place for all interactive controls.
- Reduced panel movement helps preserve spatial predictability during interaction.
- CI accessibility policy remains unchanged: AA required, AAA advisory.

- `npm run test:e2e` passed.
