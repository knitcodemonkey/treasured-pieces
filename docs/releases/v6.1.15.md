# v6.1.15

Release date: 2026-07-03

## Summary

This patch release adds live build-planning support by showing per-color usage counts for the current canvas directly in the studio UI.

## Highlights

- Added a Build Counts panel to the studio interface.
- Shows live counts for every palette color on the current canvas.
- Displays a colored-cells summary for quick material estimation.
- Keeps counts sorted by highest usage first for easier build prep.

## Included Updates

- Added Build Counts markup in `index.html`.
- Added counts panel styling in `features/studio-ui/studio-ui.css`.
- Added live color-usage calculation and render hooks in `features/studio-ui/studio-ui.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- This release adds a live informational panel only; no input interaction model changes were introduced.
