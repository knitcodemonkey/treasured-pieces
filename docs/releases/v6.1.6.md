# v6.1.6

Release date: 2026-07-02

## Summary

This patch release rebalances motif color distribution across the library, reducing heavy warm bias and broadening representation of cool and neutral palette families.

## Highlights

- Reworked existing warm-heavy motifs to include cooler and neutral variants.
- Added new single motifs focused on cool and green-toned palettes.
- Added new border motifs for night-sky and ruby-chain themes.
- Expanded usage of previously underrepresented colors, including gray, black, and red.
- Improved overall motif palette diversity while preserving motif structure and behavior.

## Included Updates

- Updated single motif definitions in features/project-core/motifs/single-motifs.js.
- Updated border motif definitions in features/project-core/motifs/border-motifs.js.
- Updated release metadata in package.json, index.html, CHANGELOG.md, and README.md.
- Added release notes entry docs/releases/v6.1.6.md and linked it from docs/releases/README.md.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Motif picker behavior and keyboard navigation remain unchanged.
- Color diversity improvements do not alter control semantics or interaction patterns.
