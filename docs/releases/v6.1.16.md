# v6.1.16

Release date: 2026-07-03

## Summary

This patch release adds restrained asymmetric accent marks to the full-window Sun and Moon cards so they feel more organic and less mechanically mirrored while keeping their symbolic tarot identity intact.

## Highlights

- Added sparse asymmetric dusk-star accents to `XVIII • The Moon`.
- Added sparse asymmetric solar accents to `XIX • The Sun`.
- Preserved the symbolic sigil-style compositions introduced in the previous release.
- Kept the accents minimal so the main arcana symbols remain the focal point.

## Included Updates

- Updated Sun and Moon full-window motif definitions in `features/project-core/motifs/single-motifs.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.

## Accessibility Compliance
