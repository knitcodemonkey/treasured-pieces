# v6.0.0

Release date: 2026-07-02

## Summary

This release introduces a native feature-sliced architecture for maintainability, with co-located code/styles/tests and a new user-flow E2E suite that validates real studio journeys.

## Highlights

- Moved to feature-sliced folders for native modular ownership.
- Removed legacy root-level app/css/js entry paths.
- Co-located core logic and tests under `features/project-core/`.
- Co-located studio behavior and styles under `features/studio-ui/`.
- Added user-flow E2E tests under `features/studio-e2e/`.
- Added deterministic local tooling (`package.json`, pinned dev dependencies).
- Added CI workflow for end-to-end studio user-flow coverage.

## Included Updates

- New core test command: `npm run test:core`.
- New E2E flow command: `npm run test:e2e`.
- Updated accessibility audit compatibility in `scripts/a11y-check.cjs`.
- Updated docs/process for co-located testing and release validation.
