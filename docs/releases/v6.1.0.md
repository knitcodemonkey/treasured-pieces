# v6.1.0

Release date: 2026-07-02

## Summary

This release introduces a new visual motif workflow that replaces dropdown-driven template loading with tappable motif previews, while keeping the iPad mini toolbar compact and feature-focused.

## Highlights

- Added visual motif picker cards with mini previews and motif names.
- Motif selection and loading are now a single tap.
- Updated toolbar structure to tabbed feature panels (Canvas, Symmetry, Motifs).
- Updated terminology from Template to Motifs in user-facing controls.

## Included Updates

- Added motif preview card rendering and active motif state in `features/studio-ui/studio-ui.js`.
- Added motif picker card styles in `features/studio-ui/studio-ui.css`.
- Updated controls markup for motif picker and tabbed panels in `index.html`.
- Updated E2E flow coverage for visual motif selection in `features/studio-e2e/studio-e2e.test.cjs`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Motif picker uses grouped button controls with active pressed state.
- Focus-visible outlines remain active on all interactive controls.
- CI accessibility policy remains unchanged: AA required, AAA advisory.

## Validation

- `npm run test:core` passed.
- `npm run test:e2e` passed.
