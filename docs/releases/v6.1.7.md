# v6.1.7

Release date: 2026-07-02

## Summary

This patch release improves clock-tower stained glass support by replacing a simple hourglass motif with a more ornate centerpiece suitable for grandfather clock window panels.

## Highlights

- Replaced the prior basic clock-hourglass pattern with an ornate "Astral Clockwork" design.
- Added vaulted crown details, jewel spokes, and pendulum-inspired lower geometry.
- Preserved centered placement behavior for compatibility across varying window widths.
- Kept color contrast and motif readability suitable for narrow, tall 17-cell panels.

## Included Updates

- Updated clock motif definition in features/project-core/motifs/single-motifs.js.
- Updated release metadata in package.json, index.html, CHANGELOG.md, and README.md.
- Added release notes entry docs/releases/v6.1.7.md and linked it from docs/releases/README.md.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Motif picker interaction and keyboard flows are unchanged.
- Update is data-level motif content only; no control semantics changed.
