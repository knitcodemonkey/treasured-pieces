# v7.2.2

Release date: 2026-07-04

## Summary

This patch release improves motif application behavior and map-art border repeat consistency by preserving interior artwork when applying borders and standardizing map-art border repeat widths for 128x128 framing workflows.

## Highlights

- Border motifs now apply non-destructively to edge regions, preserving the interior canvas.
- Added regression tests to prevent border-apply interior wipe regressions.
- Standardized map-art border repeat widths to stay within 12-20 cells.
- Ensured map-art border repeats divide 128 evenly for consistent full-frame tiling.

## Included Updates

- Updated `features/project-core/project-core.js` so `border-repeat` templates do not clear the full grid before application.
- Updated `features/project-core/project-core.test.js` with:
  - interior-preservation regression coverage for border application,
  - map-art border width/tile-divisibility assertions for 128x128.
- Updated `features/project-core/motifs/map-art-border-motifs.js` with 128-friendly repeat dimensions and refreshed repeat geometry.
- Updated release metadata in `package.json`, `index.html`, `README.md`, `CHANGELOG.md`, and `docs/releases/README.md`.
