# v6.1.17

Release date: 2026-07-03

## Summary

This patch release adjusts the lower field of the full-window Sun card so the Moon keeps ownership of the light-blue visual language while the Sun gains a stronger grounded color progression.

## Highlights

- Reworked the lower field of `XIX • The Sun` into a bottom-centered near-half-circle.
- Applied the requested color sequence:
  - `magenta`
  - `purple`
  - `blue`
  - `gray`
  - `black`
- Preserved the existing Sun sigil and asymmetric accent work.
- Kept the Moon’s `light_blue` identity visually distinct.

## Included Updates

- Updated the Sun full-window motif definition in `features/project-core/motifs/single-motifs.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.
