# v7.2.3

Release date: 2026-07-04

## Summary

This patch release adds first-class zoom controls and iPad-focused pinch zoom support so makers can comfortably work at multiple detail levels, especially on large map-art canvases.

## Highlights

- Added toolbar zoom controls with slider, increment/decrement buttons, reset, and live zoom percent.
- Added two-finger pinch zoom on canvas for touch devices, including iPad workflows.
- Kept pinch and toolbar controls synchronized through shared zoom state.
- Prevented accidental paint events while an active pinch gesture is in progress.

## Included Updates

- Updated `features/studio-ui/studio-ui.js` with zoom state management, canvas scaling integration, and touch pinch handlers.
- Updated `index.html` with new zoom controls in the top toolbar.
- Updated `features/studio-ui/studio-ui.css` with zoom control styling for desktop and touch layouts.
- Updated `features/studio-e2e/studio-e2e.test.cjs` with zoom behavior assertions (zoom-in and reset).
- Updated release metadata in `package.json`, `index.html`, `README.md`, `CHANGELOG.md`, and `docs/releases/README.md`.
