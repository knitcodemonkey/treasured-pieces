# v7.2.1

Release date: 2026-07-04

## Summary

This patch release improves border rendering consistency and visual clarity by fixing border tile orientation behavior and refining map-art border motif design quality.

## Highlights

- Fixed border-repeat rendering so side edges use rotated tile orientation.
- Ensured frame depth now reads consistently across all four edges.
- Refined map-art border motifs with cleaner, more legible large-scale patterns.
- Improved corner companion continuity so corners flow into border runs.

- Updated `features/project-core/project-core.js` border-repeat rendering to rotate tile cells on vertical edges.

- Updated release metadata in `package.json`, `index.html`, `README.md`, `CHANGELOG.md`, and `docs/releases/README.md`.
