# v7.2.9

Release date: 2026-07-04

## Summary

This patch release tightens the toolbar layout by moving Clear Canvas to the secondary row with Motifs and Symmetry, while keeping the primary row focused on Map Art and the mode-specific controls.

## Highlights

- Moved Clear Canvas out of the primary row and into the secondary toolbar row.
- Kept Motifs and Symmetry grouped together below the Map Art controls.
- Preserved the mode-dependent swap between Map Art zoom and standard canvas size controls.
- Shortened the Map Art label to reduce visual weight and improve spacing.

## Included Updates

- Updated [index.html](index.html) to group Clear Canvas with Motifs and Symmetry in the secondary controls row.
- Updated [features/studio-ui/studio-ui.css](features/studio-ui/studio-ui.css) to support the secondary-row layout.
- Preserved the existing Map Art mode swap logic in [features/studio-ui/studio-ui.js](features/studio-ui/studio-ui.js).
- Updated release metadata in [package.json](package.json), [package-lock.json](package-lock.json), [README.md](README.md), [CHANGELOG.md](CHANGELOG.md), and [docs/releases/README.md](docs/releases/README.md).
