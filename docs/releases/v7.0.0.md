# v7.0.0

Release date: 2026-07-04

## Summary

This major release launches the first Map Art foundation milestone with viewport-fit scaling and square-cell guarantees for large canvases, while preserving existing drawing behavior and enforcing mandatory test coverage.

## Highlights

- Introduced a `Map Art View` toggle in the main controls for large-grid workflows.
- Added viewport-fit canvas scaling for large canvases (including 128x128).
- Enforced square-cell rendering through one scalar runtime cell size.
- Persisted Map Art View preference and restored it on reload.
- Added dedicated E2E coverage for map-art toggle, fit behavior, square-cell invariants, and persistence.

## Included Updates

- Added Map Art control markup in `index.html`.
- Added Map Art control styling in `features/studio-ui/studio-ui.css`.
- Added Map Art state persistence, fit calculation, and runtime cell-size rendering/input behavior in `features/studio-ui/studio-ui.js`.
- Added Map Art E2E coverage in `features/studio-e2e/studio-e2e.test.cjs`.
- Added roadmap/architecture/design documentation for independent Map Art scope, milestones, and structure in `docs/roadmap.md`, `docs/architecture.md`, and `docs/design/ui.md`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.
