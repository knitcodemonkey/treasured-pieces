# v6.1.10

Release date: 2026-07-02

## Summary

This patch release strengthens visual composition and symbolic direction by fixing centered border-repeat alignment and adding a tarot-inspired motif collection grounded in project philosophy.

## Highlights

- Fixed border-repeat motif placement so edge patterns are centered by phase, reducing offset artifacts.
- Added three new philosophy-driven motifs:
  - `kintsugi-gold-seams`
  - `lotus-habitat-bloom`
  - `inner-gears-clockwork`
- Evolved Wheel of Fortune imagery into clockwork gear language with clear station and hand cues.
- Improved moon motif readability by darkening the surrounding field so white lunar elements remain high contrast.
- Updated project philosophy language to explicitly include kintsugi, lotus, stained glass, and wheel/clock symbolism.

## Included Updates

- Updated border-repeat anchor calculation in `features/project-core/project-core.js` to center pattern phases along each edge.
- Expanded motif library in `features/project-core/motifs/single-motifs.js` with new tarot/philosophy motifs.
- Added motif presence assertions in `features/project-core/project-core.test.js`.
- Updated philosophy framing in `docs/philosophy.md` and symbol summary in `README.md`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, and `docs/releases/README.md`.
