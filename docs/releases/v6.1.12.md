# v6.1.12

Release date: 2026-07-03

## Summary

This patch release improves visual clarity for tall tarot-window motifs by reducing color noise and emphasizing simple, recognizable silhouettes for low-resolution pixel canvases.

## Highlights

- Simplified tall 19x26 motif rendering for stronger at-a-glance recognition.
- Reduced palette complexity in:
  - `XVI • The Tower Reforged`
  - `XVII • The Star Lotus`
- Kept `X • Wheel Of Fortune` unchanged as the preferred composition reference.
- Shifted motif emphasis from fine detail to silhouette clarity.

## Included Updates

- Updated tall full-window motif definitions in `features/project-core/motifs/single-motifs.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- This release updates motif artwork and metadata only; no interaction model changes were introduced.
