# v6.1.20

Release date: 2026-07-03

## Summary

This patch release fixes full-window placement on even-dimension canvases and tightens Sun/Moon border rendering so 19x26 cards fill cleanly with a true one-cell outer frame.

## Highlights

- Corrected centered single-template placement by centering normalized motif bounds instead of offsetting from a fixed center cell.
- Added regression assertions for 19x26 full-window behavior so top-left and bottom-right border cells are guaranteed to be filled.
- Removed the unintended inner gray border band in the Sun/Moon full-window cards.

## Included Updates

- Updated template placement flow in `features/project-core/project-core.js`.
- Updated Sun and Moon full-window motif border logic in `features/project-core/motifs/single-motifs.js`.
- Added 19x26 full-window regression checks in `features/project-core/project-core.test.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.
