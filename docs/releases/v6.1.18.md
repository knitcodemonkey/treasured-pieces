# v6.1.18

Release date: 2026-07-03

## Summary

This patch release cleans up the full-window Moon card by reducing background competition, keeping the lunar form visually distinct from surrounding cool tones.

## Highlights

- Reduced `light_blue` usage in `XVIII • The Moon`.
- Shifted the Moon card’s broader field and halo from `light_blue` to darker `blue`.
- Swapped small Moon-side highlights and star accents to `cyan` for cleaner separation.
- Preserved the existing crescent sigil, glyph, and asymmetric star layout.

## Included Updates

- Updated the Moon full-window motif definition in `features/project-core/motifs/single-motifs.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.
