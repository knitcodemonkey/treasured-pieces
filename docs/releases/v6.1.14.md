# v6.1.14

Release date: 2026-07-03

## Summary

This patch release shifts the full-window Sun and Moon cards from literal scene language to symbolic major-arcana sigils, improving thematic clarity while preserving low-resolution readability.

## Highlights

- Rethemed `XIX • The Sun` and `XVIII • The Moon` to prioritize tarot symbology over landscape-style composition.
- Added clear geometric structure (rings, axes, and ritual marker accents) to strengthen card identity.
- Preserved the 19x26 full-window format and stable motif IDs.
- Kept `X • Wheel Of Fortune` unchanged.

## Included Updates

- Updated symbolic motif composition in `features/project-core/motifs/single-motifs.js`.
- Updated release metadata in `package.json`, `index.html`, `CHANGELOG.md`, `README.md`, and `docs/releases/README.md`.
