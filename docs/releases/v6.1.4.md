# v6.1.4

Release date: 2026-07-02

## Summary

This patch release expands and organizes the motif system by splitting motif definitions into dedicated modules, adding more motif content, and improving border-motif discoverability in the modal picker.

## Highlights

- Split motif definitions out of project core into dedicated motif files.
- Added a larger motif library across centered motifs and border motifs.
- Added additional corner-companion border sets for intentional corner matching.
- Added visual corner badges in the motif picker for quick scanning.
- Restored always-on grid default while keeping the grid toggle hidden.

## Included Updates

- Motif module split under `features/project-core/motifs/`:
  - `single-motifs.js`
  - `border-motifs.js`
  - `index.js` aggregator
- Refactored motif loading in `features/project-core/project-core.js`.
- Expanded template coverage in `features/project-core/project-core.test.js`.
- Updated motif picker metadata presentation in `features/studio-ui/studio-ui.js`.
- Added motif badge styling in `features/studio-ui/studio-ui.css`.
- Updated script load order and release metadata in `index.html`.

## Accessibility Compliance

- Standard target: WCAG 2.2.
- Existing keyboard and focus-visible interactions remain unchanged.
- Motif metadata remains text-readable and includes semantic labels for assistive technologies.
- Corner-companion indicators are presented in text and visual badge form.
