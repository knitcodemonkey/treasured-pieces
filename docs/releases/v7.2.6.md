# v7.2.6

Release date: 2026-07-04

## Summary

This patch release tightens top-toolbar mode ergonomics by making zoom map-art-exclusive and swapping zoom/size controls based on Map Art View state.

## Highlights

- Made zoom controls visible only in Map Art View.
- Kept canvas size controls visible only in standard (non-map) mode.
- Reordered controls so Map Art View appears first, with mode-specific controls immediately after it.
- Preserved expected interaction behavior with updated end-to-end checks.

## Included Updates

- Updated [features/studio-ui/studio-ui.js](features/studio-ui/studio-ui.js) to enforce map-art-only zoom behavior, including UI synchronization and zoom/pinch guards outside map mode.
- Updated [index.html](index.html) to reorder top controls and keep zoom hidden by default until Map Art View is enabled.
- Updated [features/studio-ui/studio-ui.css](features/studio-ui/studio-ui.css) to ensure hidden zoom controls remain non-displayed.
- Updated [features/studio-e2e/studio-e2e.test.cjs](features/studio-e2e/studio-e2e.test.cjs) for new zoom visibility/behavior expectations.
- Updated release metadata in [package.json](package.json), [package-lock.json](package-lock.json), [README.md](README.md), [CHANGELOG.md](CHANGELOG.md), [docs/releases/README.md](docs/releases/README.md), and [index.html](index.html).
